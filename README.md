# Colour Vocabulary Website

A simple HTML and CSS project for learning Spanish colour names.

The webpage displays several colours together with their Spanish translations and matching images.

---

## Features

- Spanish colour vocabulary
- Colour-specific headings
- External CSS stylesheet
- Images resized using CSS
- Basic use of IDs and classes

---

## What I practiced

- Linking an external CSS file
- Using CSS ID selectors
- Using CSS class selectors
- Changing text colours
- Changing font weight
- Resizing images with CSS

---

## Project structure

```text
index.html
style.css
images/
```

---

## How to run

Open `index.html` in a web browser.

- CSS3
